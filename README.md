# Gesture Play - Touchless Gaming System

## Overview

Gesture Play is a real-time gesture-controlled gaming system developed using Python. It allows users to play games such as Subway Surfers without using a keyboard or mouse by recognizing hand gestures through a webcam.

## Features

* Real-time hand tracking using MediaPipe
* Touchless game control
* Gesture-based keyboard actions
* Webcam-based interaction
* Supports games like Subway Surfers

## Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI

## How It Works

The system captures video from the webcam and detects hand landmarks using MediaPipe. Specific hand gestures are mapped to keyboard actions, enabling users to control the game without physical input devices.

## Installation

1. Clone the repository.
2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

python subway_hand_controller.py
